Here are a few links to website that I frequently use.

  • Geospatial Analysis Online is a free online book written by absolute experts in the field of geospatial analysis.
  • GDAL everything you ever wanted to know about the Geospatial Data Abstraction Library tools.
  • BBBike a very efficient OpenStreetMap data extractor for multiple formats.
  • Rseek a search engine dedicated to R statistical software related questions.
  • R for Data Science an essential online book for learning about data science and statistics in R.
  • Blogdown an online book on how to create Websites with R Markdown.
  • Bookdown an online book on how to create books and technical documents with R Markdown.
  • Leaflet for R a guide to creating interactive webmaps with Leaflet for R.
  • HOT Export Tool a site for creating OpenStreetMap exports in various file formats.
